# Video Chat Room

Веб-приложение видеочат-комнаты до 4 участников: React UI, Node.js + Socket.io signaling/chat и WebRTC mesh для аудио/видео.

## Запуск

```bash
npm install
npm run dev:server
npm run dev:client
```

По умолчанию:

- клиент: `http://localhost:5173`
- сервер: `http://localhost:3001`
- health check: `http://localhost:3001/health`

Для production-сборки:

```bash
npm run build
npm run dev:server
```

Вне `localhost` нужен HTTPS, иначе браузер заблокирует `getUserMedia`.

## Ограничения

- максимум 4 участника в комнате;
- WebRTC mesh без SFU и TURN;
- состояние комнат и история чата хранятся только в памяти сервера;
- при рестарте сервера активные комнаты теряются;
- авторизация и роли отсутствуют;
- интерфейс рассчитан на desktop viewport от 1024px.

## QA checklist

- создать комнату, скопировать ссылку и войти вторым участником;
- проверить вход третьего и четвёртого участника;
- проверить отказ пятому участнику с сообщением «Комната заполнена»;
- отправить сообщение и проверить отображение имени/времени;
- проверить системные сообщения входа/выхода;
- выключить микрофон и проверить индикатор у остальных;
- выключить камеру и проверить заглушку;
- закрыть вкладку участника и проверить освобождение слота;
- открыть ссылку после удаления комнаты и убедиться, что создаётся новая пустая комната;
- проверить отказ permissions камеры/микрофона.
